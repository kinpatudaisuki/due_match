### 概要
カードゲームの１つであるデュエル・マスターズのプレイヤーが、<br>
対戦相手やチームメイトを見つけるために、交流を行えるマッチングサイトです。<br>
URL: https://due-match.com

### 環境
#### ・開発環境
MAMP：7.0<br>
Laravel：11.26.0<br>
PHP：8.3.9<br>
Web server：Apache 2.4.58 (Unix)<br>
DB：Mysql 8.0.35

#### ・本番環境
AWS EC2<br>
OS：AmazonLinux2023<br>
Laravel：11.26.0<br>
PHP：8.3.10<br>
Web server：Apache 2.4.62 (Amazon Linux)<br>
DB：MariaDB 10.5.25

### 機能
#### ログイン機能
　・ログイン<br>
　・新規登録<br>
　・ログアウト<br>
　・アカウント削除
 
#### ・ユーザー表示機能
　・一覧表示<br>
　・詳細表示<br>
　　・在住エリア表示<br>
　　・自己紹介文表示<br>
　　・プレイ中フォーマット表示<br>
　　・評価点数表示<br>
　・ユーザー検索<br>
　　・ユーザー名検索<br>
　　・在住エリア検索<br>
　　・自己紹介文検索<br>
　　・プレイ中フォーマット検索<br>
　・ユーザー評価<br>
　　・１〜３点で評価し、累計評価数を表示
 
#### ・トーク機能　
　・トークルームの作成/検索<br>
　・文字や画像の投稿、新規メッセージのメール通知<br>
　・トークルームへ他ユーザーを招待

#### ・フレンド機能
　・一覧表示<br>
　　・フレンドになったユーザー一覧<br>
　　・自分からフレンド申請したユーザー一覧<br>
　　・フレンド申請された相手ユーザー一覧<br>
　・フレンド申請<br>
　・フレンド申請承認/拒否<br>
　・フレンド解除<br>

#### ・ブロック機能
　・ユーザーのブロック<br>
　　・メッセージの非表示、メール通知をオフ<br>
　　・フレンド申請とフレンド関係の削除<br>
　・ブロック解除

#### ・運営への問い合わせ機能
　・運営と質問者に問い合わせ作成成功のメール通知
