### ・概要
カードゲームの１つであるデュエル・マスターズのプレイヤーが、交流することのできるマッチングサイトです。

### ・環境
#### ・開発環境
MAMP：7.0<br>
Laravel：11.26.0<br>
PHP：8.3.9<br>
Web server：Apache 2.4.58 (Unix)<br>
DB：Mysql 8.0.35

#### ・本番環境
AWS EC2<br>
OS：AmazonLinux2023<br>
Laravel：11.26.0<br>
PHP：8.3.10<br>
Web server：Apache 2.4.62 (Amazon Linux)<br>
DB：MariaDB 10.5.25

### ・機能
・ログイン機能<br>
・ユーザー一覧表示機能<br>
・ユーザー検索機能<br>
・ユーザー詳細確認機能<br>
・ユーザー評価機能<br>
・ユーザーブロック機能<br>
・トーク機能<br>
・トークに新規メッセージがあることをメール通知する機能<br>
・トークに他ユーザーを招待する機能<br>
・問い合わせ機能
